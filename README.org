It only works on Mac OS.

This is a Emacs key binding (for Dvorak keyboards) addon for Firefox 57+

Scroll

| keymap   | description  |
|----------+--------------|
| C-n or n | scroll down  |
| C-t or t | scroll up    |
| C-h      | scroll right |
| C-b      | scroll left  |


Tab

| keymap  | description                   |
|---------+-------------------------------|
| M-h     | next tab                      |
| M-b     | previous tab                  |
| C-x C-h | new tab                       |
| C-x k   | close current tab             |
| t       | focus on the first input area |


Others

| keymap | description     |
|--------+-----------------|
| C-g    | focus on window |
| C-r    | refresh         |
| C-H    | history forward |
| C-B    | history back    |

These keybindings work in most webpages except pages such as 'about:xxxx'.
